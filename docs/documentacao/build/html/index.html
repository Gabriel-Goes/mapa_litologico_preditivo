Welcome to PreditorTerra’s Documentation!¶

PreditorTerra is a Python package designed for the predictive analysis of lithological and mineralogical data using artificial intelligence and remote sensing technologies.

The primary focus of this package is on creating predictive lithological maps that assist in the discovery of economically viable mineral deposits. This tool has been particularly designed for applications in geological systems, such as the Socorro-Guaxupé nappe complex, located between São Paulo and Minas Gerais, Brazil.

Our package uses supervised and unsupervised machine learning techniques, along with geophysical data, to identify mineralizing patterns, especially for minerals such as graphite. These techniques include Random Forest, Support Vector Machines, and Self-Organizing Maps (SOM), which are utilized to correlate geophysical attributes with potential mineralization zones.

Note

This package is under active development, and new features are continuously being integrated to improve its performance and applicability.
